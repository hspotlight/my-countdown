# Local development

1. Install dependencies

    ```sh
    npm install
    ```

1. Run the service

    ```sh
    npm start
    ```

1. Go to the url

    ```sh
    http://localhost:9000/.netlify/functions/event-gif?on=2022-01-01&text=Happy%20New%20Year!
    ```

    - `on` Event date
    - `text` Event title
